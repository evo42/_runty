Runty Getting started

  1. Run the runty-installer in the web page root folder
    wget http://runtyapp.org/runty-installer && php runty-installer
    or download and unzip the application
    http://runtyapp.org/runty-latest.zip
  2. Include a script-tag in your HTML pages
    <script src="/runty/app.php"></script>
  3. Define editable areas in with HTML classes & IDs
    <h1 class="runty-editable" id="my-title">Aloha World!</h1>
  4. Sign in and start editing your pages as you see them
    http://your.domain.com/runty/
