Runty. Getting started!

  1. Run the runty-installer script in the web page root folder.
    wget http://runtyapp.org/runty-installer && php runty-installer
  2. Or: Download and unzip the application to the web page root folder.
    http://runtyapp.org/runty-latest.zip
  3. Include a script-tag in your pages.
    <script src="/runty/app.php"></script>
  4. Define editable areas in your pages with HTML classes / IDs.
    <h1 class="runty-editable" id="my-title">An Awesome Title!</h1>
  5. Sign in and start editing your pages. As you see them. Navigate to:
    http://your.domain.tld/runty/
