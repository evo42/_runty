Runty App NoCMS Guide

Learn how to edit the web smart!

How To Use Runty

The Runty NoCMS App is an Open Source Software application consisting of some javascript, php and css files.
- Copy the Runty App files (.htaccess, runty, .runty) to your webserver.
- Add the CSS class
"runty-editable" to all elements you want to edit (in *.html files)
- Done. NoCMS editable ready!
- Add "?edit=on" to the URL in the browser to start editing.

Installation

  1. Download Runty App as *.zip file
  2. Extract the archive to /path/to/htdocs/. Should be a LAMP like stack. Should be accessible via web browser.
  3. Add in your HTML files to all elments you want to be editable the CSS class runty-editable.
  4. Runty App needs the following files /.htaccess and folder /.runty, /runty.
  5. All other files in the downloaded archive are for demo purposes and can be removed if you like.
  6. Use a URL like http://runtyapp.org/?edit=on to start editing (demo mode).

Minimal Demo

All the code you need for editing will be added automatically to all *.html / *.htm / *.php files in the ./ root directory of the Runty App on the web server.


			<!DOCTYPE html>
			<html>
			<head>
			<meta charset="UTF-8">
			<title>Runty App NoCMS Example</title>
			</head>
			<body>
			<h1 class="runty-editable">Runty, Edit smart!</h1>
			<p class="runty-editable">Use the power of Aloha Editor to edit the web.</p>
			<span>Runty NoCMS</span>
			</body>
			</html>

Save this code as /guides/example.html and open it in the browser. Add ?edit=on to the URL to start editing.

Everything with the class runty-editable is now editable.

Configuration

Configure Aloha Editor in /.runty/aloha-editor.js and other settings can also be found in project specific /.runty/ directory. More information later.
